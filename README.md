# gae-continuous-delivery
This repo contains the basic scaffold that automatically deploy code for Google App Engine

### Steps to run

```bash
gcloud app create 
gcloud app deploy
```

### Steps to setup

1. Fork repo
2. Setup trigger on Cloud Build (GCP)
3. Enable App Engine on settings
